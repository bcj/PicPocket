Tags

Labels applied to images in PicPocket that can be used to categorize and sort images. Tags in PicPocket are hierarchical: 'animals/birds' is a subcategory of 'animals', and 'animals/birds/duck' is a subcategoriy of both.

When searching for images based on tags, any more-specific tags will be included as well (i.e., an image tagged 'animal/bird/duck/mallard' will also show up if you search for 'animal/bird' or 'animal').

You can create tags here, but you don't need to! If you add a new tag to an image, it will be created automatically.
